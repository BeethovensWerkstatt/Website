Eine Tilgung ist (1) eine von vier Grundoperationen der [Textintervention]({% link _glossary/intervention.md %}). Zugleich bezeichnet Tilgung (2) einen von vier Variantentypen. Ebenso wie die übrigen drei Bezeichnungen für Textoperationen ([Erweiterung]({% link _glossary/einfuegung.md %}), [Ersetzung]({% link _glossary/ersetzung.md %}), [Umstellung]({% link _glossary/umstellung.md %})) besitzt auch das Begriffswort *Tilgung* eine Doppelbedeutung: Wird es als *nomen actionis* gebraucht, bezeichnet es einen Prozess (Schreib- und Textoperation); als *nomen qualitatis* bezeichnet es dagegen das durch diesen Prozess erzielte Ergebnis, ein Produkt (Variantentypus). Die vier genannten Textoperationen resp. Variantentypen lassen sich unter dem strukturalistischen Oberbegriff *Substitution* zusammenfassen.  
Innerhalb einer Werkgenese erklärt eine Tilgung ein bereits ausnotiertes Textsegment für ungültig: A –> Null. Betrifft die Tilgung ein Partitursegment, so führt dies zu einer Kürzung des ursprünglichen Textumfangs. Werden dagegen innerhalb einer einzigen Stimme oder innerhalb eines Stimmenverbunds (d. h. einer Kombination mehrerer, aber nicht aller Stimmen einer Partitur) einige Textsegmente gelöscht, wobei der Partiturkontext unverändert erhalten bleibt, so müssen die gelöschten Segmente zwingend durch Pausen ersetzt werden und umgekehrt. Dieser Kontextzwang erlaubt demnach keine ersatzlose Tilgung, sondern zieht immer eine Ersetzung nach sich, wobei der Gesamtumfang der Komposition (d. h. die Anzahl vorhandener Takte) unverändert bleibt.
Wird ein colla-parte geführter Bläsersatz oder der Vokalsatz der Originalfassung nicht in die Bearbeitungsfassung (z. B. in einen [Klavierauszug]({% link _glossary/klavierauszug.md %})) übernommen, so führt dies nicht zu einer Reduzierung des Taktumfangs, sondern zu einer Reduktion der Stimmenanzahl des gesamten Notensatzes. Strukturell bzw. formalistisch gesehen sind derartige Auslassungen von Bläser- und Vokalsatz etc. keine Tilgungen, sondern Ersetzungen, die aber – etwa im Falle eines Klavierauszugs – nicht eigens durch Pausen angezeigt werden müssen. Freilich wird eine Ersetzung eines ausdifferenzierten Textsegments (Thema, Motiv, Gegenstimme etc.) durch Pausen als strukturelle Reduktion, also als Tilgung (A –> Null) wahrgenommen.  
Bei einer [Bearbeitung]({% link _glossary/bearbeitung.md %}) eines Werkes in eine andere Fassung sind Tilgungen im engen Wortsinn nur dann möglich, wenn sie vollständige Partitursegmente betreffen. Tilgungen von Partitursegmenten erzeugen zwischen den beiden Fassungen Differenzen, die sich quantitativ in voneinander abweichenden Taktumfängen zeigen.
