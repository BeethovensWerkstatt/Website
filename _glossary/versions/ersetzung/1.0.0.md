Eine Ersetzung ist (1) eine von vier Grundoperationen der [Textintervention]({% link _glossary/intervention.md %}). Zugleich bezeichnet Ersetzung (2) einen von vier Variantentypen. Ebenso wie die übrigen drei Bezeichnungen für Textoperationen ([Tilgung]({% link _glossary/tilgung.md %}), [Erweiterung]({% link _glossary/einfuegung.md %}), [Umstellung]({% link _glossary/umstellung.md %})) besitzt auch das Begriffswort *Ersetzung* eine Doppelbedeutung: Wird es als *nomen actionis* gebraucht, bezeichnet es einen Prozess (Textoperation); als nomen qualitatis bezeichnet es dagegen das durch diesen Prozess erzielte Ergebnis, ein Produkt (Variantentypus). Die vier genannten Textoperationen resp. Variantentypen lassen sich unter dem strukturalistischen Oberbegriff *Substitution* zusammenfassen.  
Innerhalb einer Werkgenese bezweckt eine Ersetzung den Austausch eines bereits vorhandenen Textsegments (A) durch ein neues, anderes (B). Betrifft die Ersetzung ein komplettes Partitursegment, so bleibt der gesamte Textumfang entweder gleich oder er wird kürzer oder länger. Betrifft die Ersetzung jedoch nur eine einzelne Stimme oder einen Stimmenverband (d. h. eine Kombination einiger, aber nicht aller Stimmen einer Partitur), so bleibt der Taktumfang des komponierten Textes unverändert, da der Partiturkontext keine Umfangsänderung einzelner Stimmen erlaubt (Kontextzwang). In allen genannten Fällen wird das ursprüngliche Segment A durch das neue Segment B für ungültig erklärt.  
Dagegen bleiben bei einer Umarbeitung eines Werkes X in eine Parallelfassung Y alle Textsegmente, die zueinander in einem Ersetzungsverhältnis stehen, valid, denn jedes Textsegment ist in eine eigenständige Werkfassung eingebettet. Zwischen dem ursprünglichen Segment A der Fassung X und seiner Ersetzung durch B in der Fassung Y besteht eine mehr oder weniger ausgeprägte [Ähnlichkeit]({% link _glossary/aehnlichkeit.md %}), die als Varianz beschrieben werden kann.  
Alle kreativen Prozesse (einen Text verfassen, dichten, komponieren, ein Bild malen, einen Film produzieren etc.) sind mit Umarbeitungs- bzw. Änderungsmaßnahmen verbundenen, die sich als Tilgung, Ersetzung, Erweiterung oder Umstellung beschreiben lassen.
