Homogener, störungsfrei verlaufender Werktext, indem jede schaffensgenetische Spur restlos gelöscht ist. Das Ideal eines Cleartexts wird durch gedruckte Werktextausgaben eingelöst.  
Kodikologische Botschaften (als Texte les- bzw. deutbare Botschaften)
